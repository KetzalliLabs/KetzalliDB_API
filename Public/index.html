Most endpoints require Firebase authentication. Include the Firebase ID token in the Authorization header.

POST /api/auth/register

Register a new user in the database (after Firebase signup)

Request Body:

{

  "email": "string",
  "display_name": "string" (optional)

  "success": true,

POST /api/items/exercises/generate-quiz NEW

Generate a quiz with multiple exercises (supports 3 modes)

Request Body (Category Mode):

{
  "mode": "category",
  "count": 10,
  "category_id": "uuid"
}

Request Body (Mixed Mode):

{
  "mode": "mixed",
  "count": 10
}

Request Body (Custom Mode):

{
  "mode": "custom",
  "count": 5,
  "exercise_ids": ["uuid1", "uuid2", "uuid3"]
}

Response:

{
  "success": true,
  "data": {
    "questions": [
      {
        "exercise": {
          "id": "uuid",
          "category_id": "uuid",
          "category_name": "Abecedario",
          "type": "multiple_choice",
          "prompt": "Question text",
          "correct_sign_id": "uuid",
          "structure_type": null
        },
        "correct": {
          "id": "uuid",
          "name": "Correct answer",
          "description": "...",
          "image_url": "https://...",
          "video_url": "https://..."
        },
        "incorrect": [
          // Array of incorrect options
        ],
        "options": [
          // Shuffled array of all options
        ]
      }
    ]
  }
}

Mode Options:

"category" - Questions from specific category (requires category_id)
"mixed" - Random questions from all categories
"custom" - Specific exercises (requires exercise_ids array)

Count: Maximum 50 questions per quiz
